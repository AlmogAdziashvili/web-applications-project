## Requirements

* [Node.js](https://nodejs.org/) (I recommend installing with [NVM](https://github.com/nvm-sh/nvm))

## Quick Start

#### Setup

```bash
npm install
```

#### for Development

Start the client
```bash
npm run start:client
```

Start the server
```bash
npm run start:server
```

Or run client and server together
```bash
npm start
```

#### for Production

```bash
npm run build
npm start:server
```
